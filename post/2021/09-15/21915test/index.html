9/15 資訊課後小考

發表於2021-09-15|更新於2021-09-25|課後小考檢討
|字數總計:1.2k|閱讀時長:4分鐘|閱讀量:

前言

相信這堂資訊課下來大部分的同學都很茫
畢竟上次學寫程式是高一的事情了
而且還是怡松上課 估計有上跟沒上一樣
更何況卓宜青老師上課速度我覺得快得跟鬼一樣

我這篇就先針對課後小考檢討
盡量讓大家看得懂
之後可能還會再補充今天這堂課的觀念

題目檢討

第一題

下列何者不屬於 C++ 保留字

  • char
  • float
  • break
  • sum

char用來宣告字元變數
float用來宣告有小數點的變數
break他會中止包在外層的(最接近) while , do while , for , 或 switch 語句
接著執行終止的語句在下方的程式碼

第二題

何者為下列運算式表示法?

  • h+1/w/w
  • (h+1)/(w*w)
  • h+1/(w*2)
  • (h+1)/w*2

題目要問的是 h+1 除以 w 的平方
在C++裡面
除法用的是/ 乘法用的是*
而 W 的平方可以寫成 w × w 也就是w*w
然後先乘除後加減
自然就寫成 (h+1)/(w*w)

第三題

下列何者不屬於C++邏輯運算子?

  • !
  • &&
  • ||
  • ==

!、&&和||皆屬於 邏輯運算子
而==則是比較運算子

第四題

下列何種資料型態佔的記憶體最小?

  • float
  • char
  • int

char: 8位元
float: 6有效位元
int: 16位元

第五題

設x=10,y=10下列比較運算結果和者不為真?

  • x!=y
  • x*2==20
  • x==y
  • x<=y

第一個選項中的!=是≠的意思所以結果為true
第二個選項要先把 x*2 算出來 也就是 20 再去比較
比較運算子中的==是數學符號中的=
算完後 20 = 20 結果為true
第三個選項 10不等於20吧…
很明顯錯 結果為false
第四個選項 <=是數學符號中的≦
x<20 所以結果為false

第六題

下列哪一個選項會讓運算式 a%2==1 || b<0 || c>0 ?

  • a=3, b=-1, c=1
  • a=2, b=1, c=-1
  • a=3, b=-1, c=-2
  • a=3, b=4, c=5

首先我們來看看or的真值表
|x|否|是|
|:—:|:—:|:—:|
|否|否|是|
|是|是|是|
可以發現OR只要一個值為是其他輸出的都會是是
那這就簡單了 我們只要找出那個全部都不正確的就好了
每個選項都帶進去後會發現 只有第二個選項全錯

第七題

以下和著為「被2整除，但不能被3整除的所有整數n」的條件式?

  • n%2 == 0 && n%3 != 0
  • n/2 == 0 && n/3 != 0
  • n%2 == 0 || n%3 != 0
  • n/2 == 0 || n/3 != 0

如何判斷是否被整除?
看有沒有餘數 我們都知道C++裡面的%就是取餘數的
被2整除指的是n%2 == 0
而被3整除指的是n%3 == 0
所以被2整除，但不能被3整除就是n%2 == 0 && n%3 != 0

第八題

下列程式碼執行的結果?

這題基本上順著程式碼就可以算出a/b
只不過要注意的是第8行cout<<"a/b="<<a/b<<endl;
在打出 a/b 的結果之前要先打a/b=再打 a/b 的結果

第九題

下列程式碼執行的結果?

這一題除了跟上一題一樣的小細節以外
要注意一下程式碼的流程
在第6行的時候x已經被賦予y的值了 也就是 20
所以在下一行y賦予 的值時y被賦予的就是 20

第十題

下列程式碼執行的結果?

這一題…
第六行的時候z被賦予的x的值 x z都是2
下一行 x被賦予了y的值也就是2
此時的x=2,y=2,z=1
第八行 最終y被賦予了z的值
所以此時x=2,y=1
仔細看的話
這個題目其實是在告訴你程式碼如何把兩個變數的值對調
高一時教的氣泡排序法就有使用到這段

最後

這次檢討就到這邊啦
雖然估計不會有什麼人看

有問題的話我能幫到的我會盡量幫
但不要像盛文考試問我答案:(
